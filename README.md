## Build Promela SPIN from Source on Ubuntu 14.04 (Freya)

* http://spinroot.com/spin/Src/index.html
* sudo apt-get install bison
* tar zxvf spin643.tar.gz
* cd Spin
* make
* ./spin

Spin Version 6.4.3 -- 16 December 2014
pre-built => /toolkit/spin
